Aobe WorkTrack Portable 1.0.0

Run Aobe-WorkTrack-Portable-1.0.0.exe directly. No installer, Node.js, Python,
Rust, database software, or command prompt is required. Windows WebView2 is
required and is already included with current Windows 10 and Windows 11 systems.

The portable executable still stores its private SQLite database in the current
Windows user's application-data folder. Moving the executable does not move the
records. Use Settings > Backup & import > Back Up Now to move records safely.
